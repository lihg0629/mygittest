## 1. 常用命令

## 1-1. init

> 初始化本地仓库

```bash
git init 
git init [directory]
git init [directory] [-b branchname | --initial-branch=branchname]
```



## 1-2. add 

```bash
git add .
git add [filename]
```



## 1-3. commit

```bash
git commit -m 'comment'
```



## 1-4. remote

```bash
git remote add origin <url>
git push origin master
```



## 1-3. clone 

```bash
git clone [-b branchname] <repo> [directory]
```



## 1-4. origin

这就是个默认名字 ，可以变。
